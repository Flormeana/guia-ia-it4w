Sección 01 — Guía IA IT4W

Los modos de trabajo:
Ask, Plan y Agent

Cuando abrís el chat de Copilot en VS Code, podés elegir cómo querés que la IA trabaje con vos. Hay tres modos, cada uno con un nivel distinto de autonomía.

Pensalo como pedirle algo a un asistente:
Ask = "¿me explicás esto?"  ·  Plan = "¿cómo lo harías vos?"  ·  Agent = "hacelo vos, yo te sigo."
🔍
Ask
Para explorar y entender

Hacés una pregunta, la IA responde. No toca nada, no modifica nada. Solo te explica o te orienta.

Ejemplo IT4W "¿Qué archivos tiene este proyecto y para qué sirve cada uno?"
🗺️
Plan
Para pensar antes de hacer

Le pedís a la IA que te proponga cómo abordar una tarea. Te da una hoja de ruta, no una acción.

Ejemplo IT4W "Planificá cómo escribirías la documentación de este flujo de pruebas"
🤖
Agent
Para ejecutar y delegar

La IA no solo responde: toma decisiones, usa herramientas y ejecuta tareas por vos.

Ejemplo IT4W "/generar-seccion concepto: KPIs de ventas archivo: contenido/kpis.md"

¿Cómo se usa?

  1. Abrí el panel de chat de Copilot en VS Code
  2. Seleccioná el modo: Ask, Plan o Agent
  3. Escribí tu pedido en lenguaje natural — no necesitás saber programar
  4. Leé la respuesta y seguí desde ahí
💡 Tip: Si no estás segura de cuál usar, empezá con Ask. Siempre podés cambiar de modo.
