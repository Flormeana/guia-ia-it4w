Sección 02 — Guía IA IT4W

GitHub Copilot vs Claude:
el mismo lugar, distinto motor

En VS Code podés usar distintos modelos de IA dentro del mismo chat de Copilot. GitHub Copilot es el asistente de Microsoft. Claude es el modelo de Anthropic. Ambos viven en el mismo panel, pero piensan distinto.

Es como elegir entre dos médicos especialistas: los dos son buenos, pero uno tiene más experiencia en lo que necesitás.
Microsoft
GitHub Copilot
Modelo base integrado
  • Especializado en código y completado de texto
  • Sugerencias mientras escribís
  • Integración nativa con VS Code
  • Respuesta rápida para tareas cortas
Anthropic
Claude Sonnet
Modelo para lenguaje y flujos
  • Especializado en razonamiento y lenguaje natural
  • Entiende contexto largo y documentos complejos
  • Ideal para redactar, revisar y coordinar
  • Lee y aplica skills con precisión

Ejemplo real — Sofía, diseñadora UX

La tarea
"Redactá las instrucciones de flujo de pantallas para el cliente."
❌ Con Copilot base: Respuesta genérica, en inglés, pensada para código.
✅ Con Claude Sonnet: Lee el archivo de instrucciones del proyecto, entiende el tono IT4W y redacta directo en español con el estilo del equipo.

¿Cómo cambio el modelo?

  1. Abrí el chat de Copilot en VS Code
  2. Hacé clic en el selector de modelo (arriba en el panel, muestra el nombre actual)
  3. Elegí el modelo: GPT-4o, Claude Sonnet, etc.
  4. Escribí tu pedido normalmente — el resto es igual
