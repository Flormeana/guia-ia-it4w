# Los modelos de IA: Copilot y Claude

Cuando hablás con la IA en VS Code, hay algo funcionando "adentro" que procesa lo que escribís y genera la respuesta. Ese algo se llama modelo. Podés elegir cuál usar.

---

> "No es Copilot VS Claude. Usan el mismo lugar, cambia el motor."

---

## La analogía del auto

Imaginá un auto que te deja elegir entre distintos motores antes de salir. El volante, los pedales y el tablero son siempre los mismos. Lo que cambia es el rendimiento y las características.

VS Code es el auto. GitHub Copilot y Claude son los motores.

---

## GitHub Copilot

Es el modelo "de base" — el que viene activo por defecto cuando instalás la extensión. Lo desarrollan GitHub y OpenAI juntos.

Es muy bueno para tareas estructuradas: revisar archivos, analizar código, razonamiento paso a paso.

**Mejor para:**
- Explorar y entender la estructura de un proyecto
- Razonamiento lógico y análisis
- Respuestas rápidas y directas

---

## Claude

Claude lo desarrolla Anthropic. Es el modelo alternativo que podés elegir desde el selector de VS Code.

Es especialmente bueno para escribir en lenguaje humano: documentación, comunicaciones, guías, textos que van a leer personas reales.

**Mejor para:**
- Redactar comunicados internos y documentación
- Generar contenido con tono específico (cálido, formal, simple)
- Tareas creativas y de escritura larga

---

## ¿Cuándo uso cuál en IT4W?

| Tarea | Copilot | Claude |
|---|---|---|
| Entender la estructura del proyecto | ✅ Muy bueno | ✅ Muy bueno |
| Redactar un comunicado interno | Puede servir | ✅ Ideal |
| Generar documentación para usuarios | Puede servir | ✅ Ideal |
| Revisar archivos y resumir | ✅ Muy bueno | ✅ Muy bueno |
| Escribir presentaciones y pitches | Regular | ✅ Ideal |

En esta guía y en los proyectos IT4W usamos **Claude** como modelo principal para las tareas de redacción y documentación.

---

## ¿Cómo cambio el modelo?

En el chat de Copilot, fijate en el selector que está arriba del cuadro de texto. Generalmente muestra el nombre del modelo activo. Hacé clic en él y elegí el modelo que querés usar.

El cambio se aplica al próximo mensaje que enviés.
