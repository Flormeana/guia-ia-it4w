# Instalación: lo que necesitás antes de empezar

Necesitás tres cosas. Cada una tiene un paso claro. Tardás menos de 10 minutos en total.

---

## 1. VS Code

VS Code es el programa donde vas a trabajar. Es gratuito.

**Descargalo acá:**
🔗 [https://code.visualstudio.com/download](https://code.visualstudio.com/download)

Elegí la versión para Windows (o Mac si usás Mac). Instalalo como cualquier programa: siguiente, siguiente, instalar.

Cuando abras VS Code por primera vez, vas a ver una pantalla oscura con texto. Es normal. No tocás nada todavía.

---

## 2. Cuenta de GitHub

GitHub es el servicio que le da acceso a la IA. **IT4W ya te la provee** — no tenés que crearla vos.

Si todavía no recibiste tus credenciales de GitHub, escribile a tu líder de área o al equipo de Adopción IA. Te las mandan en minutos.

---

## 3. Extensión GitHub Copilot

La extensión es el puente entre VS Code y la IA. Se instala desde adentro del programa.

**Pasos:**

**Paso 1.** Abrí VS Code.

**Paso 2.** En la barra lateral izquierda, buscá el ícono de cuatro cuadraditos (se llama "Extensions"). Hacé clic ahí.

**Paso 3.** En el buscador que aparece arriba, escribí:
```
GitHub Copilot
```

**Paso 4.** El primer resultado que aparece se llama "GitHub Copilot" y tiene el logo de GitHub. Hacé clic en el botón azul que dice **Install**.

**Paso 5.** Cuando termine de instalar, VS Code te va a pedir que inicies sesión con tu cuenta de GitHub. Hacé clic en **Sign in** y usá las credenciales que te dio IT4W.

**Paso 6.** Una vez que iniciaste sesión, vas a ver un ícono nuevo en la barra lateral izquierda — parece un globo de chat. Ese es el chat de Copilot. Está listo para usar.

---

## ¿Cómo sé que funcionó?

Si ves el ícono del chat en la barra lateral y podés hacer clic en él, funcionó.

Si algo no anduvo, escribí en el canal de Teams **Adopción IA — Dudas y Consultas**. El equipo responde rápido.
