# VS Code: tu oficina digital

En el ejercicio anterior usaste VS Code sin saber bien qué era. Ahora que lo usaste, tiene más sentido explicarlo.

---

> "VS Code es tu oficina digital. Los archivos son tu escritorio. La IA es tu asistente."

---

## ¿Qué es VS Code?

VS Code (Visual Studio Code) es un programa para editar archivos de texto. Eso es todo.

Suena simple porque lo es. Lo que lo hace especial es que fue diseñado para ser extensible: podés agregarle herramientas, conectarlo con servicios externos y — en nuestro caso — integrarlo con una IA que trabaja dentro del mismo programa.

No es solo para programadores. Cualquier persona que trabaje con archivos de texto — documentación, comunicaciones, guías, análisis — puede usarlo.

---

## El mapa de la interfaz

Cuando tenés VS Code abierto con una carpeta de proyecto, la pantalla tiene estas zonas:

**Barra lateral (izquierda)**
La columna de íconos que viste en el ejercicio. Cada ícono abre un panel diferente:
- 📁 **Explorer** — muestra todos los archivos de tu proyecto
- 🔍 **Search** — busca texto dentro de todos los archivos
- 🔀 **Source Control** — muestra los cambios de Git (lo que cambiaste pero no guardaste)
- 💬 **Copilot Chat** — el chat con la IA

**Editor (centro)**
La zona grande donde se abren y editan los archivos. Si abrís el archivo `notas.md` que creaste antes, el texto aparece acá.

**Terminal (abajo)**
La consola de comandos. La abrís con `Ctrl + ñ`. Acá escribís los comandos de Git y otros comandos del sistema.

**Barra de estado (abajo del todo)**
Una barra azul o violeta con información: en qué línea estás, qué tipo de archivo tenés abierto, si hay errores.

---

## La analogía de la oficina

Pensalo como tu espacio físico de trabajo:

- **La carpeta del proyecto** = el cajón del escritorio para ese proyecto específico
- **Los archivos** = los documentos de ese cajón
- **El editor** = la mesa donde abrís un documento y lo editás
- **La terminal** = el teléfono para mandar órdenes al sistema
- **El chat de Copilot** = tu asistente que está sentado al lado y puede hacer cosas por vos

---

## ¿Por qué no usar Google Docs o Word?

VS Code permite trabajar con archivos de texto plano (`.md`, `.txt`, `.html`, etc.) que la IA puede leer, modificar y crear directamente. Google Docs y Word usan formatos propietarios que la IA no puede tocar con la misma libertad.

Además, VS Code tiene acceso a Git, a la terminal y a extensiones — todo en el mismo lugar.
