# Glosario: todos los términos de la guía

Si te encontraste con una palabra que no entendiste, acá está la definición simple. Ordenado alfabéticamente.

---

| Término | ¿Qué es? | Analogía |
|---|---|---|
| **Agent** | Modo de trabajo en el que la IA ejecuta acciones reales: crea archivos, modifica contenido, invoca subagentes. | Tu asistente al que le delegás una tarea y la hace solo. |
| **Agente** | Una IA configurada con instrucciones específicas para hacer una tarea determinada. Técnicamente, es un archivo `.md` con esas instrucciones. | Un empleado especializado con su manual de trabajo. |
| **Agente principal** | El agente que recibe tus pedidos, lee las instrucciones del proyecto y coordina a los subagentes. | El director de orquesta. |
| **Ask** | Modo de trabajo en el que la IA responde preguntas sin tocar ningún archivo ni ejecutar nada. | Preguntarle algo a un compañero. Te responde, no hace nada. |
| **Claude** | Modelo de IA desarrollado por Anthropic, disponible en VS Code como alternativa a GitHub Copilot. Muy bueno para redacción y comunicación. | El motor premium del auto, optimizado para escritura. |
| **Commit** | Una "foto" guardada del estado de tu proyecto en un momento determinado. | Sacar y etiquetar una foto del proyecto. |
| **Copilot Chat** | El panel de chat integrado en VS Code donde hablás con la IA. | La ventana de mensajes con tu asistente de IA. |
| **Extensión** | Un complemento que le agrega funcionalidades a VS Code. GitHub Copilot es una extensión. | Un accesorio que le agregás a un aparato para que haga más cosas. |
| **Git** | Sistema de control de versiones que guarda el historial de cambios de tu proyecto dentro de tu propia carpeta. | El álbum de fotos del proyecto, guardado en tu cajón. |
| **git add .** | Comando que prepara todos los archivos nuevos o modificados para el próximo commit. | Encuadrar la foto antes de sacarla. |
| **git commit** | Comando que guarda el estado actual del proyecto con un nombre/descripción. | Sacar y etiquetar la foto. |
| **git init** | Comando que activa Git en una carpeta por primera vez. | Activar la cámara. |
| **git log** | Comando que muestra el historial de commits. | Ver el álbum de fotos del proyecto. |
| **GitHub** | Servicio web donde se pueden alojar proyectos que usan Git para compartirlos con otros. No es lo mismo que Git. | La galería de fotos en línea donde subís el álbum. |
| **GitHub Copilot** | La extensión de VS Code que conecta el editor con la IA. También es el nombre del modelo de IA por defecto. | El cable que conecta tu programa con el asistente de IA. |
| **IT4W** | Empresa donde trabajan los usuarios de esta guía. | — |
| **MCP** | Model Context Protocol. Sistema que permite que la IA se conecte con herramientas externas como Figma o GitHub desde VS Code. | El teléfono que le permite a la IA hablar con otras oficinas. |
| **Modelo** | El motor de inteligencia artificial que procesa lo que escribís y genera las respuestas. Podés elegir entre GitHub Copilot y Claude. | El motor del auto. El volante y los pedales no cambian. |
| **Modo** | La forma en que trabaja la IA: Ask (responde), Plan (propone), Agent (ejecuta). | Las marchas de un auto: primera, segunda, tercera. |
| **Plan** | Modo de trabajo en el que la IA propone una hoja de ruta para una tarea, sin ejecutar nada. | Pedirle a alguien que te cuente cómo haría algo antes de hacerlo. |
| **Proyecto** | En VS Code, una carpeta que abrís en el programa. Todo lo que está dentro de esa carpeta es tu proyecto. | Un cajón del escritorio dedicado a un tema. |
| **RepoAdopcion** | El repositorio oficial de IT4W donde se centraliza todo el material de la iniciativa IA 2026. | La biblioteca del equipo de Adopción IA. |
| **Skill** | Un archivo de texto con instrucciones especializadas que le explicás a la IA cómo hacer una tarea. | El manual de procedimientos del especialista. |
| **Subagente** | Un agente especializado en una sola tarea, que el agente principal llama cuando lo necesita. | El especialista al que llama el director cuando lo necesita. |
| **Terminal** | Ventana de texto dentro de VS Code donde escribís comandos. Se abre con Ctrl + ñ. | El teléfono interno del programa para mandar órdenes al sistema. |
| **VS Code** | Visual Studio Code. El programa donde trabajás con la IA en IT4W. Es gratuito y funciona con extensiones. | Tu oficina digital. |
| **#file:** | Sintaxis para adjuntar un archivo al chat de Copilot. Se usa para cargar skills y dar contexto adicional. | Adjuntar un documento a un mensaje. |
