# Abrí VS Code: lo que ves por primera vez

Abriste VS Code. Hay texto, íconos, paneles. No se parece a Word ni a Google Docs. No importa — en dos minutos vas a saber dónde está cada cosa.

---

## El mapa de la pantalla

Cuando abrís VS Code, la pantalla tiene cuatro zonas principales:

```
┌─────────────────────────────────────────────────┐
│  BARRA LATERAL   │   EDITOR (la zona de trabajo) │
│  (íconos a la    │                               │
│   izquierda)     │   Acá abrís y editás archivos │
│                  │                               │
│  📁 Explorer     │                               │
│  🔍 Buscar       │                               │
│  🔀 Git          ├───────────────────────────────┤
│  💬 Copilot      │   TERMINAL                    │
│                  │   (la consola de comandos)    │
└─────────────────────────────────────────────────┘
```

---

## ¿Qué es cada zona?

**La barra lateral izquierda — los íconos**
Son accesos directos. Los más importantes:
- El ícono de carpetas (📁) = el Explorador. Te muestra todos tus archivos.
- El ícono de globo de chat (💬) = el chat de Copilot. Acá hablás con la IA.

**El editor — la zona grande del centro**
Es donde se abren los archivos. Si abrís un Word, acá ves el texto. Si abrís una foto, la ves acá. Si no abriste ningún archivo, está vacío.

**La terminal — abajo**
Es una ventana donde escribís comandos. Parece complicada pero vas a usar solo tres comandos siempre iguales. La abrís con `Ctrl + ñ` (o Ctrl + backtick en teclados en inglés).

---

## Lo que probablemente ves la primera vez

Si acabás de instalar VS Code, lo más probable es que veas una pantalla de bienvenida con links y opciones. Podés cerrarla — hacé clic en la X de la pestaña que dice "Welcome".

Si te quedó la pantalla oscura y vacía: perfecto. Eso es lo que queremos.

---

## Una sola cosa para recordar

No tenés que memorizar todo esto. Solo necesitás saber dónde está el **chat de Copilot** (el ícono de globo en la barra izquierda) y dónde está el **Explorador de archivos** (el ícono de carpeta, también en la barra izquierda).

Lo demás lo vas aprendiendo a medida que lo usás.
