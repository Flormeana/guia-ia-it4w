# Crear tu primera carpeta de proyecto

En VS Code, trabajás con carpetas, no con archivos sueltos. La carpeta que abrís se convierte en tu proyecto. Todo lo que crees adentro está organizado, junto, y es tuyo.

Es como tener un cajón en el escritorio dedicado exclusivamente a un proyecto. Todo lo que sea de ese proyecto va ahí adentro.

---

## Paso 1. Abrí el Explorador de Windows

Apretá las teclas `Windows + E` al mismo tiempo. Se abre el Explorador de archivos — la ventana que usás para ver tus documentos, descargas y escritorio.

---

## Paso 2. Navegá hasta el Escritorio

En el panel izquierdo del Explorador de Windows, buscá y hacé clic en **Escritorio**.

---

## Paso 3. Creá una carpeta nueva

Hacé clic derecho en un espacio vacío del Escritorio → **Nueva** → **Carpeta**.

Escribí este nombre exacto:
```
mi-primer-proyecto-ia
```

Apretá Enter. La carpeta apareció en el Escritorio.

---

## Paso 4. Abrí esa carpeta en VS Code

Ahora vas a decirle a VS Code que esa carpeta es tu proyecto.

**Opción A — desde VS Code:**
- Abrí VS Code
- En el menú superior, hacé clic en **File** → **Open Folder...**
- Buscá la carpeta `mi-primer-proyecto-ia` que creaste en el Escritorio
- Hacé clic en ella para seleccionarla → hacé clic en **Seleccionar carpeta**

**Opción B — arrastrando:**
- Arrastrá la carpeta desde el Escritorio y soltala encima de la ventana de VS Code.

---

## ¿Qué pasó?

Cuando VS Code abre una carpeta, la muestra en el panel de la izquierda (el Explorador). Por ahora está vacía — acabás de crear el proyecto.

Fijate arriba del todo en VS Code: aparece el nombre `MI-PRIMER-PROYECTO-IA` en letras mayúsculas. Eso confirma que estás adentro del proyecto correcto.

---

## Lo importante

Esta carpeta ES tu proyecto. Todo lo que crees acá adentro — archivos, notas, documentos — va a quedar organizado junto. Cuando guardes el proyecto con Git (lo hacemos en unos pasos), Git va a guardar la historia de todo lo que hay en esta carpeta.
