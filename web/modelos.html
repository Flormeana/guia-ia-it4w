Sección 02

Los modelos de IA:
Copilot y Claude

📖 Concepto

¿Qué es un modelo de IA?

Cuando le escribís algo a la IA en VS Code, hay un "motor" que procesa tu mensaje y genera la respuesta. Ese motor se llama modelo. Podés elegir cuál usar — y cada uno tiene una personalidad distinta.

"No es Copilot VS Claude. Usan el mismo lugar, cambia el motor."
🤖
GitHub + OpenAI
GitHub Copilot
El modelo que viene activo por defecto. Preciso, estructurado, muy bueno para analizar y razonar paso a paso.

MEJOR PARA:

  • Analizar archivos y estructuras
  • Razonamiento lógico
  • Respuestas técnicas y directas
✨
Anthropic
Claude
El modelo alternativo, especialmente bueno para escribir en lenguaje humano: documentación, comunicaciones, explicaciones para personas reales.

MEJOR PARA:

  • Redactar con tono específico
  • Documentación para no-técnicos
  • Textos largos y creativos
💡 Ejemplo cotidiano

Dos traductores, el mismo escritorio

Imaginá que tenés dos traductores en la misma oficina, los dos trabajan en tu computadora:

TraductorEspecialidadCuándo llamarlo
CopilotDocumentos técnicos, contratos, especificacionesCuando necesitás precisión y estructura
ClaudeComunicación, cartas, textos para el públicoCuando necesitás claridad y tono humano

Los dos están en el mismo lugar. Solo cambiás el selector.

💡 Para tener en cuenta — Copilot en VS Code vs Copilot CLI

Dos formas de usar Copilot — no son lo mismo

Además del chat dentro de VS Code, existe Copilot CLI — una versión de Copilot que funciona fuera del editor, en una ventana de comandos separada. Son complementarias, no excluyentes.

💻 Copilot en VS Code
  • Visual, interactivo, paso a paso
  • Ves el historial de conversación fácilmente
  • Ideal para explorar, ajustar y trabajar en el día a día
  • Consume más RAM y CPU — puede hacer lento el editor en proyectos grandes
⚡ Copilot CLI
  • Corre fuera del editor — usa muy poca RAM y CPU
  • Autónomo: delegás tareas grandes y él las ejecuta solo
  • Ideal cuando VS Code se pone lento o el proyecto es muy grande
  • No tiene vista simple del historial para el equipo
✅ Recomendación del equipo IT4W

Usá los dos según la situación: VS Code para pensar, explorar y hacer ajustes del día a día. CLI para delegar trabajo largo o cuando el editor empieza a ir lento. No tenés que elegir uno — se complementan.

✅ Checkpoint

Lo que aprendiste en esta sección

  • 🤖 Copilot y Claude viven en el mismo lugar — el chat de VS Code.
  • 🔄 Cambiás de modelo con el selector que está abajo del cuadro de texto.
  • 📐 Copilot: más preciso para análisis y estructura.
  • ✍️ Claude: más conversacional para redacción y comunicación.
  • 💻 Copilot en VS Code es visual e interactivo, pero usa más recursos.
  • ⚡ Copilot CLI es liviano y autónomo — ideal para tareas grandes o cuando el editor va lento.
  • ✅ Tenés un archivo comparacion.md con ambas respuestas para volver a consultar.
