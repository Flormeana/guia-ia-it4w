Sección 02

Los modelos de IA:
Copilot y Claude

Cuando hablás con la IA en VS Code, hay algo funcionando "adentro" que procesa lo que escribís y genera la respuesta. Ese algo se llama modelo. Podés elegir cuál usar.

"No es Copilot VS Claude. Usan el mismo lugar, cambia el motor."
Imaginá un auto que te deja elegir entre distintos motores antes de salir. El volante, los pedales y el tablero son siempre los mismos. Lo que cambia es el rendimiento. VS Code es el auto. Copilot y Claude son los motores.

¿Qué es cada uno?

🤖
Desarrollado por GitHub + OpenAI
GitHub Copilot
Es el modelo "de base" — el que viene activo por defecto cuando instalás la extensión. Muy bueno para tareas estructuradas: revisar archivos, analizar código, razonamiento paso a paso.

MEJOR PARA:

  • Explorar y entender la estructura de un proyecto
  • Razonamiento lógico y análisis
  • Respuestas rápidas y directas
✨
Desarrollado por Anthropic
Claude
Claude es el modelo alternativo que podés elegir desde el selector de VS Code. Especialmente bueno para escribir en lenguaje humano: documentación, comunicaciones, guías, textos para personas reales.

MEJOR PARA:

  • Redactar comunicados internos y documentación
  • Generar contenido con tono específico
  • Tareas creativas y de escritura larga

💡 En esta guía y en los proyectos IT4W usamos Claude como modelo principal para las tareas de redacción y documentación.

¿Cómo cambio el modelo?

  1. Abrí el chat de Copilot en VS Code
  2. Fijate en el selector que está arriba del cuadro de texto — muestra el nombre del modelo activo
  3. Hacé clic en él y elegí el modelo que querés usar
  4. El cambio se aplica al próximo mensaje que enviés
