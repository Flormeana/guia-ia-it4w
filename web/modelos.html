Sección 02

GitHub Copilot y Claude:
dos motores, un mismo lugar

Cuando abrís VS Code, vas a poder elegir qué modelo de IA quiere usar. Los modelos son distintos — cada uno tiene características y fortalezas. Lo importante: los operás desde el mismo lugar.

"El mismo lugar, distinto motor."
Es como un auto que te deja elegir entre nafta super o premium — el volante y los pedales no cambian, pero el rendimiento sí.

¿Qué es cada uno?

🤖
Desarrollado por GitHub + OpenAI
GitHub Copilot
Es el modelo "de base" de VS Code. Muy bueno para tareas técnicas, autocompletar código y razonamiento estructurado. Es el que viene por defecto.

Mejor para:

  • Revisar archivos y resumir estructuras
  • Razonamiento paso a paso
  • Buscar respuestas rápidas
✨
Desarrollado por Anthropic
Claude
Es el modelo alternativo disponible desde VS Code. Excelente para redacción, documentación, ideas creativas y comunicaciones más complejas.

Mejor para:

  • Redactar textos para personas reales
  • Documentación interna y guías
  • Pensar comunicaciones complejas

Ejemplo práctico: Sofía de UX

Sofía necesita documentar el flujo de onboarding de un nuevo usuario para la App de IT4W.

❌ Sin elegir el modelo correcto
Sofía usa el modelo por defecto y le pide que explique el flujo técnico de la API. La IA le da una respuesta muy técnica que no sirve para el documento de diseño que necesita.
✅ Con Claude seleccionado
Sofía cambia a Claude y pide el mismo documento en tono accesible para stakeholders. La respuesta tiene tono claro, estructura de secciones y lenguaje para no-técnicos.
💡 En la guía IT4W usamos Claude como modelo principal porque está optimizado para redacción, documentación y comunicación — las tareas más comunes del equipo.

¿Cómo se cambia el modelo?

  1. Abrí el chat de Copilot en VS Code
  2. En la parte superior del chat hay un selector de modelo
  3. Hacé clic y elegí Claude Sonnet (o el modelo disponible en tu cuenta)
  4. Tu próxima pregunta ya usa el modelo nuevo
