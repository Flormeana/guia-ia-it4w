Sección 01

Los modos de trabajo:
Ask, Plan y Agent

Cuando abrís el chat de GitHub Copilot en VS Code, podés elegir cómo querés que la IA trabaje con vos. Hay tres modos, cada uno con un nivel distinto de autonomía.

"VS Code es la oficina. La IA es el equipo que trabaja adentro."
Pensalo como pedirle algo a un asistente: Ask = "¿me explicás esto?"  ·  Plan = "¿cómo lo harías vos?"  ·  Agent = "hacelo vos, yo te sigo."

¿Para qué sirve cada uno?

🔍
Para explorar y entender
Ask
Hacés una pregunta, la IA responde. No toca nada, no modifica nada. Solo te explica o te orienta.

Cuándo usarlo:

  • Cuando querés entender antes de actuar
  • Para que te resuma un documento o proyecto
  • Cuando no sabés ni por dónde empezar
Ejemplo UX "¿Qué archivos tiene este proyecto y para qué sirve cada uno?"
🗺️
Para pensar antes de hacer
Plan
Le pedís a la IA que te proponga cómo abordar una tarea. Te da una hoja de ruta, no una acción.

Cuándo usarlo:

  • Antes de encarar algo nuevo o complejo
  • Para entender qué pasos hay antes de ejecutar
  • Para validar un enfoque antes de actuar
Ejemplo QA "Planificá cómo escribirías la documentación de este flujo de pruebas."
🤖
Para ejecutar y delegar
Agent
La IA toma decisiones, usa herramientas y ejecuta tareas por vos. Lee archivos, crea documentos y encadena acciones.

Cuándo usarlo:

  • Cuando ya sabés qué querés y querés que la IA lo haga
  • Para tareas de varios pasos coordinados
  • Para flujos completos: generar → revisar → diseñar
Ejemplo Dashboards "/generar-seccion concepto: KPIs de ventas archivo: contenido/kpis.md"

¿Cómo se usa?

  1. Abrí el panel de chat de Copilot en VS Code (ícono de chat a la izquierda)
  2. En la parte superior del chat, seleccioná el modo: Ask, Plan o Agent
  3. Escribí tu pedido en lenguaje natural — sin necesitar saber programar
  4. Leé la respuesta y seguí desde ahí
💡 Si no estás segura de cuál usar, empezá con Ask. Siempre podés cambiar de modo.
