Sección 01

Los modos de trabajo:
Ask, Plan y Agent

En el ejercicio viste la diferencia entre Ask y Agent. Ahora entendemos los tres modos con más detalle — y cuándo usar cada uno en el trabajo real de IT4W.

"VS Code es la oficina. La IA es el equipo que trabaja adentro."

Los tres modos

🔍
Para explorar y entender
Ask
Ask es para preguntar. La IA responde, explica, orienta. No toca ningún archivo. No ejecuta nada. Solo te da información.

CUÁNDO USARLO EN IT4W:

  • Cuando querés entender qué hace un archivo o carpeta
  • Cuando no sabés por dónde empezar con una tarea
  • Cuando querés que te resuma algo antes de decidir si lo hacés
Ejemplo "¿Qué archivos tiene este proyecto y para qué sirve cada carpeta?"
🗺️
Para pensar antes de actuar
Plan
Plan es para que la IA te proponga una hoja de ruta. Te dice cómo haría la tarea — pero vos decidís si seguir ese plan o no. No ejecuta nada todavía.

CUÁNDO USARLO EN IT4W:

  • Antes de una tarea nueva o compleja
  • Cuando querés ver el camino completo antes de dar el primer paso
  • Cuando querés validar un enfoque antes de invertir tiempo en él
Ejemplo "Quiero documentar el flujo de onboarding de nuevos usuarios. ¿Cómo lo encarás?"
🤖
Para que las cosas pasen
Agent
Agent es el modo de ejecución. La IA toma decisiones, usa herramientas, crea archivos y encadena acciones. Es el modo donde realmente se hace el trabajo.

CUÁNDO USARLO EN IT4W:

  • Cuando ya sabés qué querés y querés que la IA lo haga
  • Para tareas de múltiples pasos que se pueden delegar
  • Para invocar subagentes especializados
Ejemplo "/generar-seccion concepto: 'Qué es un skill' archivo: contenido/skills.md"

⚠️ Nota sobre versiones nuevas: En versiones recientes de VS Code y GitHub Copilot, puede que no veas los mismos nombres exactos. El modo "Edit" ya no está disponible — fue reemplazado por Agent. Si en tu versión ves solo Ask y Agent, no te preocupés: el flujo es el mismo.
💡 La regla práctica: Cuando no sabés cuál usar, empezá con Ask. Te orienta sin hacer nada. Si querés que algo pase de verdad, pasá a Agent.
