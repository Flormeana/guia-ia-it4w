Ejercicio práctico · Parte 1

Tu primer día con IA:
de cero a tu primer proyecto

Este ejercicio te lleva de la mano desde instalar VS Code hasta tener tu primer proyecto funcionando con Copilot + Claude. Son 6 pasos que podés hacer en orden, sin saltarte ninguno.

💡 Si ya tenés VS Code instalado, podés arrancar desde el paso 2.

1
Instalación
Instalá VS Code
Tener el editor donde vive la IA

VS Code es el editor de texto donde vive Copilot y Claude. Es gratuito y lo usan tanto diseñadores como desarrolladores.

SistemaPasos
WindowsDescargá el instalador desde code.visualstudio.com y ejecutalo
MacDescargá el .zip desde code.visualstudio.com, descomprimí y mové a Aplicaciones

Al terminar la instalación, abrí VS Code. Deberías ver la pantalla de bienvenida.

2
Explorador de archivos
Creá tu carpeta de trabajo
Un lugar organizado para todo lo que vas a crear con IA

Primero creá la carpeta desde tu computadora, como lo harías con cualquier otra carpeta. No necesitás abrir ningún programa especial todavía.

🪟 En Windows
  1. Abrí el Explorador de archivos (la carpeta amarilla en la barra de tareas)
  2. Navegá al lugar donde querés guardar tu proyecto (Escritorio, Documentos, etc.)
  3. Click derecho en un espacio vacío → Nuevo → Carpeta
  4. Escribí el nombre que quieras, sin espacios — por ejemplo: mi-proyecto-ia
🍎 En Mac
  1. Abrí el Finder (el ícono de cara en el Dock)
  2. Navegá al lugar donde querés guardar tu proyecto
  3. Menú Archivo → Nueva carpeta, o Cmd + Shift + N
  4. Escribí el nombre que quieras, sin espacios — por ejemplo: mi-proyecto-ia

Ahora abrí esa carpeta en VS Code:

A
La forma más fácil: Dentro de VS Code, andá al menú File → Open Folder… → buscá tu carpeta → click en "Seleccionar carpeta"
B
Arrastrando: Arrastrá la carpeta desde el Explorador (Windows) o Finder (Mac) directo a la ventana de VS Code

VS Code se va a reabrir mostrando tu carpeta vacía en el panel izquierdo.

3
VS Code — Extensions
Instalá Copilot
La extensión que trae la IA a VS Code

Copilot es una extensión — un complemento que se agrega a VS Code para darle acceso a la IA.

Cómo abrir ExtensionsPasos
Con atajosWindows: Ctrl + Shift + X / Mac: Cmd + Shift + X
Con el íconoClick en el ícono de bloques (cuarto ícono del panel izquierdo)
Por menúView → Extensions

En el buscador de Extensions, escribí "GitHub Copilot". Instalá la extensión con publisher GitHub (la primera que aparece). Cuando termine, VS Code te va a pedir que inicies sesión con tu cuenta de GitHub de IT4W.

⚠️ Usá siempre tu cuenta de GitHub corporativa de IT4W, no una cuenta personal.
4
Chat de Copilot — modo Ask
Tu primera conversación con la IA
Ver que funciona y probar los tres modos

Abrí el chat de Copilot:

Cómo abrirloPasos
Con atajosWindows: Ctrl + Alt + I / Mac: Cmd + Option + I
Por menúView → GitHub Copilot
Con el íconoClick en el ícono de Copilot (panel derecho)

Asegurate de estar en modo Ask (el selector arriba del chat). Escribí:

Hola, soy nuevo en VS Code. ¿Podés explicarme en 3 oraciones qué podemos hacer juntos?

Cuando la IA responda, probá cambiar al modo Agent y hacé la misma pregunta. Notá si la respuesta cambia.

💡 ¿La respuesta te confundió? Pedile que la simplifique: "Explicalo de vuelta pero como si yo tuviera 12 años."
5
VS Code — Source Control
Guardá tu progreso con Git
No perder lo que creaste — el sistema de guardado profesional

Git es el sistema que guarda el historial completo de cambios de tu proyecto. En VS Code está integrado.

⚠️ Git es diferente a guardar con Ctrl+S. Ctrl+S guarda el archivo. Git guarda un punto de control en la historia del proyecto — podés volver a cualquier versión anterior.

No necesitás saber nada de Git ni de comandos. Pedile al agente que lo haga por vos. Abrí el chat de Copilot en modo Agent y copiá esto:

💬 Copiá esto en el chat — modo Agent 
Inicializá Git en este proyecto si no está inicializado, y hacé el primer commit con el mensaje "Mi primer proyecto con IA". Si Git te pide mi nombre y email, usá [tu nombre] y [tu email de IT4W].

El agente ejecuta los tres pasos por vos: inicializar, agregar los archivos y guardar el primer punto de control.

6
¡Completado!
¡Listo! Tu entorno de IA está funcionando
Verificar que todo está en orden

Ya tenés todo configurado. Revisá que se cumplan estos puntos:

¿Qué tenés?¿Cómo verificarlo?
✅ VS Code instaladoAbrís VS Code sin problemas
✅ Carpeta de proyectoLa ves en el panel izquierdo de VS Code
✅ Copilot instalado y autenticadoPodés abrir el chat y recibir respuestas
✅ Primera conversación hechaLa IA te respondió coherentemente
✅ Git inicializadoVes el ícono de Source Control en el panel con "0 changes"

¿Todo en orden? Ahora sí podés explorar el resto de la guía para sacarle el máximo a la IA en tu trabajo diario.
