Tu primer día con IA en VS Code

Antes de leer sobre modos, modelos y agentes — hacelo. Este ejercicio te lleva desde cero hasta guardar tu primer trabajo con IA. Tiempo estimado: 20 minutos.

"Aprendés a usar la IA usándola, no leyendo sobre ella."

1
Orientación · 3 min
Abrí VS Code
Conocer la pantalla antes de tocar nada

Abrí VS Code. Si no lo tenés instalado, pedíselo a tu líder de área — viene con GitHub Copilot ya incluido para los equipos IT4W.

Cuando VS Code abre, vas a ver esto:

ZonaQué es
Barra izquierda (íconos)Accesos: archivos, búsqueda, extensiones, chat de IA
Panel central grandeAcá se abren y editan los archivos
Panel inferiorTerminal — donde escribís comandos como git
Ícono de chat (burbuja)GitHub Copilot — la IA que vas a usar
💡 No te preocupés si no entendés todo todavía. En los próximos pasos vas a usar cada zona.
2
Configuración · 3 min
Creá tu carpeta de proyecto
Tener un lugar donde guardar tu trabajo

Antes de empezar, necesitás un lugar donde guardar todo lo que vas a crear. Esa carpeta ES tu proyecto.

  1. Abrí el explorador de archivos:
    • Forma tradicional: click derecho en el Escritorio → "Abrir en el Explorador de archivos"
    • Windows: Windows + E
    • Mac: click en Finder en el Dock, o Cmd + Space → escribí "Finder"
  2. Navegá a una ubicación que conozcas (por ejemplo, tu escritorio o Documentos)
  3. Creá una carpeta nueva llamada mi-primer-proyecto-ia
  4. Volvé a VS Code y abrí la carpeta:
    • Forma tradicional: menú Archivo → Abrir carpeta
    • Windows: Ctrl + K, luego Ctrl + O
    • Mac: Cmd + K, luego Cmd + O
  5. Seleccioná la carpeta que acabás de crear

VS Code ahora está "parado" en tu carpeta. Todo lo que hagás va a quedar ahí.

💡 El nombre de la carpeta no tiene espacios — eso es para que Git y la terminal funcionen sin problemas.
3
Configuración · 3 min
Abrí GitHub Copilot
Activar la IA y elegir el modo correcto

GitHub Copilot es la IA integrada en VS Code. Para abrirla:

FormaCómo hacerlo
TradicionalClick en el ícono del chat (burbuja de diálogo) en la barra izquierda
WindowsCtrl + Alt + I
MacCmd + Option + I

Se abre el panel de chat. Podés escribir ahí directamente.

En la parte superior del chat vas a ver un selector de modo. Hay tres opciones:

ModoEn una oración
AskLa IA responde y explica. No toca nada.
PlanLa IA propone cómo hacer algo. Vos decidís.
AgentLa IA actúa: crea archivos, ejecuta pasos.

Para este paso, elegí Ask.

4
Ask y Agent · 5 min
Tu primera conversación con la IA
Sentir la diferencia entre pedir y hacer

Primero, con el modo Ask, escribí esto:

Estoy empezando a usar VS Code con IA. ¿Qué debería tener en mi primera carpeta de proyecto?

La IA te va a responder con sugerencias. No crea nada todavía, solo te informa.

Ahora cambiá a modo Agent y mandá el mismo mensaje. Observá la diferencia:

ModoQué pasa
AskLa IA sugiere qué hacer. Vos lo hacés si querés.
AgentLa IA propone actuar directamente — puede crear archivos.
⚠️ En modo Agent, la IA puede pedirte confirmación antes de crear archivos. Leé antes de confirmar.
💡 Ahora ya viviste la diferencia entre los modos. Eso es lo más importante de este paso.
5
Terminal · Git · 5 min
Guardá tu trabajo con Git
Crear el historial de tu proyecto

Git guarda la historia de tu proyecto — como un álbum de fotos de todo lo que hiciste. Primero, abrí la terminal de VS Code:

FormaCómo hacerlo
TradicionalMenú Terminal → New Terminal
WindowsCtrl + Ñ  o  Ctrl + `
MacCmd + `

Con la terminal abierta, ejecutá estos tres comandos, uno por vez:

git init
git add .
git commit -m "primer commit: comienzo del proyecto"
ComandoAnalogía
git initAbrís el álbum de fotos del proyecto
git add .Seleccionás todas las fotos del rollo para revelar
git commit -m "..."Revelás las fotos y las ponés en el álbum con una etiqueta

Si no hay errores, tu proyecto quedó guardado. Verificalo con:

git log --oneline
